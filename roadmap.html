Q2 2020 - Stealth Prototype

The Stealth Prototype was released in Q2 2020.

This release focussed on a Prototype implementation that builds up a Proof of Concept with the core UI/UX of the Browser.

Q1 2021 - Stealth Private Alpha

The Private Alpha of Stealth will be relased in Q1 2021.

This release will focus on the core UI/UX of the Browser in order to enable a safe way to filter out malicious parts of the Web by modifying the content before it reaches the User-Agent.

The following features are going to be part of it:

  • Site Optimizers to allow optimization of content
  • CSS Parser to filter out privacy-compromising CSS
  • HTML Parser to filter out privacy-compromising HTML
  • Beacons to allow extraction of content
  • stealth:history Page
  • stealth:media Page
  • stealth:search Page

Q3 2021 - Radar Private Alpha

The Private Alpha of Radar will be released in Q3 2021.

Radar will be the Knowledge Tracker that allows to share Beacons and Echoes with other Stealth instances on the Web.

The following features are going to be part of it:

  • Global Censorship Index (by Country and ISP)
  • Sharing of Beacons with connected Peers
  • Sharing of Echoes with connected Peers
  • Sharing of Hosts with connected Peers (as a decentralized cache for DNS)

Q4 2021 - Radar Public Alpha

The Public Alpha of Radar will be released in Q4 2021.

This release will focus on Security Enhancements by implementing an automated way to distribute Intermediary Certificates that allow peer-to-peer TLS encryption.

The following features are going to be part of it:

  • Root Certificate for radar.tholian.network
  • Cross-Signing Web Service for [username].radar.tholian.network
  • Registration Service for Users (based on HOTP session tokens and local private keys)
  • Global Index for most popular Beacons
  • Global Index for most popular Echoes
  • Global Index for similar Sites (based on Beacons and content)

Q2 2022 - Stealth Public Beta

The Public Beta of Stealth will be released in Q1 2022.

This release will focus on Site Optimizers and Echoes in order to allow users to record and share their interactions and navigations on Sites.

The following features are going to be part of it:

  • Recording of Echoes via User Interface
  • Sharing of Echoes with connected Peers
  • Site Exporters to exports of a Site's data (via CSV, JSON or XML)
  • Image Optimizers (via optipng, libjpeg-turbo, convert)
  • Video Optimizers (via ffmpeg)
  • Better Site Optimizers to auto-upgrade HTML5 content
